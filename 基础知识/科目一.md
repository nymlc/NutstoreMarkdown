扣留机动车

+ 未放置检验合格标志、保险标志
+ 上道路行驶的机动车未悬挂机动车号牌
+ 未随车携带行驶证、驾驶证
+ 伪造、变造或者使用伪造、变造的机动车登记证书、号牌、行驶证、驾驶证
+ 使用其他车辆的机动车登记证书、号牌、行驶证、检验合格标志、保险标志
+ 机动车所有人、管理人未按照国家规定投保机动车第三者责任强制保险





# 吊销机动车驾驶证

+ 驾驶拼装的机动车或已达到报废标准的机动车上路行驶的
+ 将机动车交由未取得机动车驾驶证或者机动车驾驶证被吊销、暂扣的人驾驶的
+ 机动车行驶超过规定时速百分50%的
+ 造成交通事故逃逸
+ 饮酒、醉酒后驾驶机动车的
+ 在一个记分周期内累积记分达到12分的

# 20~200

+ 持有大型客车、牵引车、城市公交车、中型客车、大型货车驾驶证的驾驶人联系电话、从业单位等信息发生变化未及时申报变更信息的
+ 实习期内驾驶机动车未按规定粘贴、悬挂实习标志的
+ 驾驶人在实习期内单独驾驶机动车上高速公路行驶
+ 机动车驾驶人补领机动车驾驶证后，继续使用原机动车驾驶证的

+ 故意遮挡、污损机动车号牌

# 200

+ 对应当自行撤离现场而未撤离的，交通警察应当责令当事人撤离现场；造成交通堵塞的

# 200~500

+ 驾驶人没有按照规定参加审验仍驾驶机动车

# 200~1000

+ 驾驶证超有效期
+ 扣12分

# 200~2000

+ 驾驶拼装的机动车或已达到报废标准的机动车上路行驶的
+ 机动车行驶超过规定时速50%的
+ 非法安装警报器、标志灯具
+ 造成交通事故后逃逸，尚不构成犯罪的
+ 未取得机动车驾驶证、机动车驾驶证被吊销或者机动车驾驶证被暂扣期间驾驶机动车的
+ 将机动车交由未取得机动车驾驶证或者机动车驾驶证被吊销、暂扣的人驾驶的










+ 强迫机动车驾驶人违反道路交通安全法律、法规和机动车安全驾驶要求驾驶机动车，造成交通事故，尚不构成犯罪的
+ 违反交通管制的规定强行通行，不听劝阻的
+ 故意损毁、移动、涂改交通设施，造成危害后果，尚不构成犯罪的
+ 非法拦截、扣留机动车辆，不听劝阻，造成交通严重阻塞或者较大财产损失的

# 1000~2000

+ 饮酒后驾驶机动车

# 1000~3000

+ 伪造、变造或者使用伪造、变造的检验合格标志、保险标志的

# 2000~5000

+ 伪造、变造或者使用伪造、变造的机动车登记证书、号牌、行驶证、驾驶证



# 十五日以下拘留

+ 未取得机动车驾驶证、机动车驾驶证被吊销或者机动车驾驶证被暂扣期间驾驶机动车的
+ 造成交通事故后逃逸，尚不构成犯罪的
+ 强迫机动车驾驶人违反道路交通安全法律、法规和机动车安全驾驶要求驾驶机动车，造成交通事故，尚不构成犯罪的
+ 非法拦截、扣留机动车辆，不听劝阻，造成交通严重阻塞或者较大财产损失的
+ 伪造、变造或者使用伪造、变造的机动车登记证书、号牌、行驶证、驾驶证

# 三年以下有期徒刑或者拘役

+ 违反交通运输管理法规，因而发生重大事故，致人重伤、死亡或者使公私财产遭受重大损失

# 三年以上七年以下有期徒刑

+ 交通运输肇事后逃逸或者有其他特别恶劣情节
+ 先死后逃

# 七年以上有期徒刑

+ 逃逸致人死亡

# 拘役

+ 在道路上驾驶机动车追逐竞驶，情节恶劣的
+ 在道路上醉酒驾驶机动车的


# 12分

+	超速50%
+ 驾驶证或车辆：故意遮挡号牌；驾驶证与车辆不符；伪造、更改号牌和证件
+ 超员：客车超员20%以上
+ 严重情形：饮酒驾驶；事故后逃逸；严重超速
+ 高速公路：营运客车在高速公路内停车；车辆在高速公路上倒车、逆行等行为
+ 其他：疲劳驾驶、未取得校车驾驶资格证

# 6分

+ 驾驶证：驾驶证暂扣；隐瞒、欺骗手段补领驾驶证。
+ 超速：驾驶中型以上载客载货汽车、校车、危险物品运输汽车，在高速路、城市快速路超速未达20%；其他车辆超速达20%-50%。
+ 超员：营运客车、校车超员未达20%；其他车辆超员20%以上。
+ 超载：货车超载30%以上。
+ 危险物品：危险物品未按指定时间、路线、速度行驶的。
+ 违反信号灯

+ 普通道路逆行

+ 打电话

+ 驾驶证转到异地或者有效期满换证时，应当到公安交管部门接受审验
